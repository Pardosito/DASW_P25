GitHub es una plataforma en línea para alojar, compartir y colaborar en proyectos de desarrollo de software utilizando Git, un sistema de control de versiones. Permite a los desarrolladores trabajar juntos en código, hacer seguimieparanto de cambios, gestionar versiones y contribuir a proyectos de código abierto o privados.

Algunos de sus comandos principales son:

  • git init
  • git clone URL_DEL_REPO
  • git add .
  • git commit -m "Mensaje descriptivo"
  • git push origin main
  • git pull origin main
  • git branch NUEVA_RAMA
  • git checkout NUEVA_RAMA
  • git merge OTRA_RAMA
